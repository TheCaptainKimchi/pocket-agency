# Assets Directory

This directory contains game assets for Pocket Agency.

## Structure

- `/sprites` - Character sprites, UI elements, and game objects
- `/tiles` - Tileset images for map building
- `/sounds` - Sound effects and background music

## Placeholder Files

Currently contains placeholder files for testing. Replace with actual game assets during development.

## Asset Guidelines

### Sprites
- Format: PNG with transparency
- Recommended size: 32x32 or 64x64 pixels
- Naming convention: `object_name_state.png` (e.g., `agent_idle.png`)

### Tiles
- Format: PNG
- Recommended size: 16x16 or 32x32 pixels
- Can be organized as tilesets

### Sounds
- Format: MP3 or OGG
- Keep file sizes reasonable for web/mobile
- Naming convention: `sound_name.mp3` (e.g., `mission_complete.mp3`)
